README

Gloze requires java 5.0, and jena-2.6.4 (installed by ant from <http://jena.sourceforge.net/>)

Gloze Project Properties -> Java Build Path -> Source, should be set to:
Gloze/src/com
Gloze/src/beaninfo
Gloze/test/src

Documentation generated by Doxygen <www.doxygen.org> may be found at doc/html/index.html

To generate Gloze documentation with ant you need to install and run Doxygen with the Doxyfile

To test Gloze with ant you need to execute ant -f test.xml

The samples folder contains standalone samples and scripts.
